# Reports 文档入口

本目录存放本地验收和专项验证报告，适合需要确认某项能力是否已经被本地验证时查看。

## 命名说明

- `*_LOCAL_CHECK.md`：单项功能或场景的本地验收记录
- 报告通常对应某个功能、某次回归或某一类稳定性验证

## 常见使用方式

- 想确认某功能是否验收过：直接按文件名搜索关键词
- 想补齐回归留痕：参考已有报告格式新增同类文档
- 想追踪某项能力入口：从 `guides/PLAYER_FEATURES_USAGE_VALIDATION.md` 反查到对应报告
