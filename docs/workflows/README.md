# Workflows 文档入口

本目录存放操作手册、检查清单、回归流程和 AI 阅读剧本，适合需要照流程执行或系统化阅读代码时查看。

## 适合什么时候看

- 需要做格式回归或功能回归
- 需要遵循当前团队工作流
- 需要用 AI 辅助阅读源码、分模块分析代码
