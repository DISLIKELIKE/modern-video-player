# MPC-HC 功能差距评估（截至 2026-03-07）

## 1. 评估口径

本评估用于回答“当前项目距离 MPC-HC 级别播放器还有多少功能未实现”。

评估口径：
- 只按“端到端可用”计为已实现（主流程可触达、可运行、可被用户使用）。
- 只有类/接口/占位代码但未接入主播放链路，计为“骨架/未接入”。
- 参考基线为桌面播放器常见能力集合（MPC-HC 同类能力），不是学习型最小播放器目标。

代码核对范围（关键入口）：
- `src/main.cpp`
- `src/video_player.cpp`
- `src/core/player_core.cpp`
- `src/display.cpp`
- `src/render/*`
- `src/subtitle/*`
- `src/playlist/*`
- `src/streaming/*`
- `src/plugin/*`

## 2. 当前已具备的可用能力

当前可稳定使用的能力主要是“基础本地播放链路”：
- 本地媒体打开、解封装、音视频解码与播放。
- 播放控制：播放、暂停、停止、拖拽进度 seek。
- 基础交互：全屏、音量调节、简单控制条覆盖层。
- 播放调度：独立解码线程和渲染线程，含基础背压与丢帧策略。
- 日志与基础配置能力（播放器日志链路可用）。

## 3. 差距总览（按 14 个模块）

| 模块 | 当前状态 | 结论 |
|---|---|---|
| 01 核心播放引擎 | 已可播放，但仍是基础版 | 部分实现 |
| 02 滤镜系统 | 管道和内置滤镜有实现，但缺少用户侧接入 | 部分实现 |
| 03 视频渲染器 | SDL 可用，D3D11/OpenGL 仍为占位 | 部分实现 |
| 04 音频渲染器 | SDL 音频可用，均衡器/混音未接入播放主链 | 部分实现 |
| 05 字幕系统 | SRT 解析器存在，未渲染、未同步、未接入主链 | 骨架/未接入 |
| 06 播放列表 | M3U8 管理类可用，未接入播放器流程和 UI | 部分实现 |
| 07 解码器管理 | 能力探测/选择代码存在，未驱动真实硬解链路 | 骨架/未接入 |
| 08 文件格式支持 | 扩展名能力声明存在，依赖 FFmpeg 基础能力 | 部分实现 |
| 09 流媒体支持 | HLS/DASH 解析器为轻量实现，下载器未实现读流 | 骨架/未接入 |
| 10 皮肤系统 | 仅主题变量容器 | 骨架/未接入 |
| 11 快捷键系统 | HotkeyManager 存在但未接入事件主链 | 骨架/未接入 |
| 12 设置系统 | INI 读写可用，但未接入播放器运行配置 | 部分实现 |
| 13 插件系统 | 仅内存注册/启停状态，无动态加载与生命周期 | 骨架/未接入 |
| 14 视频增强 | 亮度/对比度/饱和度算法在库中，未形成可控产品能力 | 部分实现 |

模块统计：
- 达到 MPC-HC 等级：`0 / 14`
- 部分实现：`8 / 14`
- 骨架或未接入：`6 / 14`

粗略结论：
- 当前对齐度约 `25%~35%`
- 尚未实现或未打通的能力约 `65%~75%`

## 4. 关键未实现功能清单

### P0（先补，直接影响“能不能当主力播放器”）

- 多音轨切换、字幕轨切换、外部字幕加载与同步显示。
- 完整字幕渲染链（至少 SRT 端到端，建议补 ASS/SSA）。
- Windows 硬解通路（D3D11VA）端到端打通，而不是仅能力声明。
- D3D11/OpenGL 渲染器真正可用并可切换。
- 播放速度控制端到端生效（含音频时钟/音频变速策略）。
- 播放列表与播放器状态机打通（连续播放、上一首/下一首、历史）。
- 设置系统接入（启动加载、退出保存、运行期应用）。

### P1（补齐体验，接近 MPC-HC 常用操作）

- 章节导航、A-B Repeat、循环模式。
- 帧步进、截图、缩放/画幅比切换、旋转。
- 音轨延迟/字幕延迟调节。
- 可配置快捷键并持久化。
- 渲染器/解码器/音频输出设备可选项。

### P2（平台化与扩展能力）

- 插件系统：动态加载、版本兼容、生命周期、错误隔离。
- 流媒体：真实 HTTP 分片下载、缓冲管理、自适应码率。
- 皮肤系统：主题资源化与组件级样式覆盖。

## 5. 代码层证据摘要

以下文件可直接说明当前“已实现 vs 骨架”的边界：
- 播放主链：`src/core/player_core.cpp`
- SDL 渲染主路径：`src/render/sdl_video_renderer.cpp`
- D3D11 占位：`src/render/d3d11_video_renderer.cpp`
- OpenGL 占位：`src/render/opengl_video_renderer.cpp`
- 字幕仅解析未接入：`src/subtitle/srt_parser.cpp`
- 流媒体下载未实现：`src/streaming/http_stream_downloader.cpp`
- 插件仅内存注册：`src/plugin/plugin_manager.cpp`
- 设置系统未接入主链：`src/config/settings_manager.cpp`

## 6. 建议里程碑（面向落地）

里程碑 1（先“可用”）：
- 打通字幕显示、播放列表串联、设置加载保存、播放速度端到端。

里程碑 2（再“好用”）：
- 打通 D3D11VA + D3D11 渲染路径，补齐常用播放控制（章节、截图、帧步进）。

里程碑 3（再“可扩展”）：
- 插件系统与流媒体完整化，形成接近 MPC-HC 的长期演进基础。
